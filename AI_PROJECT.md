# Project Overview
Tabi-log MVP (personal travel album app)

## Tech Stack
- Expo (React Native)
- TypeScript
- Expo Router
- Supabase (later)

## Development Policy
This is MVP only.
Keep implementation minimal.
Do NOT add extra features.

## MVP Features

### 1. Navigation Tabs
- Home
- Quest (Explore)
- Album

### 2. Album (local only for now)
- Create album (title only)
- Album list
- No cloud yet

### 3. Quest
- Static list (mock data)

### 4. Home
- Simple welcome screen

## NOT in MVP
- Authentication
- Payments
- Image upload
- Supabase integration (later)
- Notifications
- Animations
- UI design improvements

## Coding Rules
- TypeScript strict
- Functional components only
- No any
- No new libraries unless instructed

## Do NOT touch
- node_modules
- package-lock.json
- .expo
- build
